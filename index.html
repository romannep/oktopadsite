Description

Tasks

The task has a date and time when you need to start its execution. The task can be quickly rescheduled for several hours ahead. It is possible to create recurring tasks.

Calendar

For scheduling and organizing appointments use Google Calendar. It is convenient to see events for the day next to the tasks. In the application, you can connect your Google Calendar to see business day meetings.

List of changes

  • Tasks with repeat
  • View calendar events

Privacy policy

The Oktopad application uses Google authentication to access your calendars to work with it's events according to your intentions and show them for you. Oktopad application neither store internaly no send somewhere any information from you Google account, including data from Calendar.

Описание

Задачи

У задачи указывается дата и время, когда надо приступить к ее выполнению. Задачу можно быстро перенести на несколько часов вперед. Есть возможность создания периодических задач.

Календарь

Для планирования и организации встреч используйте Google Calendar. События на текущий день удобно видеть в рядом с задачами. В приложении вы можете подключить свой Google Calendar чтобы видеть встречи на текущий день.

Список изменений

  • Задачи с возможностью повтора
  • Просмотр событий календаря

Политика конфиденциальности

Приложение Oktopad использует аутентификацию Google для доступа к вашим календарям, чтобы работать с событиями в соответствии с вашими намерениями и показывать их для вас. Приложение Oktopad не хранит внутри себя и не отправляет куда-либо информацию из вашей учетной записи Google, в том числе данные из Календаря.
